New Features!

Richer tuple types

Extracting and spreading parameter lists

Rest parameters can now be defined as a generic and inferred as tuples!

See example tuple-rest.ts

Richer tuple types


				type Coordinate = [ number, number, number? ];
				type Restful = [ number, ... string[]];
				type Empty = [];

tuple types better match parameter lists

The unknown type

the least-capable type

See example unknown.ts
  • Requires guards or type casting
  • Is great for user-provided data
  • any allows anything, unknown requires identification
